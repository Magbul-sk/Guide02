GIT-HUB

let's learn how to push the code in GIT-HUB

  • NOTE : the main file should be as index.html

  • Search github.com in Browser.
  • Sign in
  • Open your Profile and choose Repositories
  • Now give the Name & Description
  • Make it as public & Add README file
  • Create Repository
  • Add files & choose/drag your files later commit changes
  • then it will process the files
  • later go to settings , select pages left side
  • change branch none to main and save it
  • then go to code part & it will take 3 min to verify
  • if verification is done in pages it will generate the link
  • through that link you can visite your website project
